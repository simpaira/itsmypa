"Unsigned app? Why should I trust you?"

Because you don't have to. The entire source code is public. Read it, build it yourself, or watch its network traffic — after the one-time model download, there isn't any. A $99 Apple certificate proves someone paid Apple. Open source proves what the app actually does.

Questions, answered

Is it really free? What's the catch?

Really free, open source, no account. If it takes off we may add an optional paid tier for power features — the core private notetaker stays free. You can also support development on GitHub.

Does anything ever leave my Mac?

No. The one exception is the first launch, which downloads the AI models (~5 GB, one time) from public model repositories. After that, everything — recording, transcription, summaries, chat — runs locally. Airplane mode works.

Why does macOS warn me when I install it?

We haven't paid Apple's $99/year code-signing fee yet. The app isn't malware — the full source is public, which proves more than a certificate does. System Settings → Privacy & Security → "Open Anyway", or install via Homebrew. Signing is planned.

What are the requirements?

An Apple Silicon Mac (M1 or newer), macOS 13+, 16 GB RAM recommended, and ~6 GB of disk for the app and models.

Why is Screen Recording permission needed?

macOS gates system-audio capture (the other people in your call) behind the Screen Recording permission. ItsMyPA never captures or stores your screen — only the audio. Skip it and recording still works, mic-only.

Which languages does it support?

Whisper supports ~100 languages. English is strongest; for non-Latin-script languages (Hindi, Arabic, Chinese, Japanese…), switch to a larger Whisper model in Settings for noticeably better accuracy.
